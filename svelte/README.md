# Svelte Dapp templates

This subdirectory provides access to Svelte dapp samples that run on the Internet Computer platform.

- [SvelteKit starter](./sveltekit-starter/README.md): A prerendered SvelteKit dapp example with a Motoko backend
- [Svelte Motoko starter](./svelte-motoko-starter/README.md): A Svelte single page app example that uses Internet Identity and a Motoko backend
